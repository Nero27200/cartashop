# CartaShop - Gestion de Collection Panini Adrenalyn XL

**CartaShop** est une application web moderne et completement optimisee pour gerer votre collection de cartes Panini Adrenalyn XL Ligue 1 2025-26 (540 cartes disponibles).

## Caracteristiques Principales

### Boutique
- 540 cartes disponibles
- Recherche en temps reel par nom, club ou position
- Filtrage par club, type et poste
- Tri par prix croissant/decroissant
- Statistiques instantanees

### Ma Collection
- Gestion personnelle de vos cartes
- Affichage des cartes possedees
- Persistance automatique via LocalStorage

### Cartes a Vendre  
- Ajoutez vos cartes a la vente
- Gerez vos prix individuellement
- Suivi d'inventaire automatique

### Panier et Achat
- Ajoutez/supprimez des cartes facilement
- Calcul automatique du total
- Visualisation claire du panier

## Technologies

- **HTML5** - Structure semantique
- **CSS3** - Design responsive + animations
- **JavaScript Vanilla** - Aucune dependance externe
- **LocalStorage** - Persistance des donnees

## Optimisations

- Debounce (300ms) pour recherche performante
- Lazy loading des images
- Animations CSS GPU-accelerees
- Code sans dependances externes

## Demarrage

1. Clonez le repository
2. Ouvrez `index.html` dans votre navigateur
3. Les cartes se chargent automatiquement depuis `cards.json`

## Structure
